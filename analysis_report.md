# Sales Data Analysis Report

## Overview
This report analyzes a simple sales dataset using Python and the pandas library.

## Dataset Description
The dataset contains information about products, quantity sold, price, and total sales.

## Analysis Performed
- Loaded CSV data using pandas
- Checked dataset structure and data types
- Handled missing values
- Calculated total sales
- Calculated average sales
- Identified the best-selling product

## Results
- **Total Sales:** Calculated by summing the Total_Sales column
- **Average Sales:** Mean of Total_Sales
- **Best-Selling Product:** Product with highest total sales

## Conclusion
The analysis helps understand overall sales performance and identifies the top-performing product using basic data analysis techniques.
